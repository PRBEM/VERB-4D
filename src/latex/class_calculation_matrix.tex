\subsection{Detailed Description}
A matrix used primarily for diagonalized calculations. 

Model matrix (or related matrices) It is based on \hyperlink{class_diag_matrix}{Diag\+Matrix} and have methods for conversion from 3\+D or 2\+D P\+S\+D (and related) arrays into 1d array of unknown elements
