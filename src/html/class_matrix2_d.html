T** Matrix2D< T >::matrix_array

Matrix array (array of links to other arrays). Final links pointed to the memory addresses of the plane array. Matrix[x][y] can be used. Also, all rows saved in the memory one after another as a big array. So Matrix[x+x_size*y] can be also used.
